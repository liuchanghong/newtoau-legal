プライバシーポリシー

施行日：2026年5月26日

AusRights は、オーストラリアの法律および公式ガイダンスに関する一般的な情報を提供するためのアプリです。本アプリは、個人情報を収集、送信、販売、または共有しません。

本ページは英語版プライバシーポリシーの翻訳です。翻訳と英語版の解釈に相違がある場合は、英語版が優先されます。

本ポリシーについて

本プライバシーポリシーは、AusRights モバイルアプリケーション（「AusRights」、「本アプリ」または「当方」）が、本アプリの利用時に情報をどのように取り扱うかを説明するものです。

収集する情報

AusRights はアカウントの作成を必要とせず、利用者から個人情報を収集しません。本アプリは、広告、分析、またはトラッキング技術を使用しません。

端末に保存される情報

表示言語を選択すると、本アプリは選択内容を記憶するため、その言語設定を端末内に保存します。本アプリがこの設定を当方または第三者に送信することはありません。設定はいつでも変更でき、本アプリを削除するとローカルに保存されたデータも削除されます。

外部ウェブサイトおよび電話

AusRights には、オーストラリア政府および関係機関の公式ウェブサイトへのリンクと、関連サービスの電話番号が含まれています。リンクを開く、または電話をかけることを選択した場合、その操作は AusRights の外部で行われ、該当するウェブサイト、サービス提供者、または通信事業者のプライバシー慣行が適用されます。AusRights は、それらの操作に関する情報を受け取りません。

本プライバシーポリシーのウェブページ

本ポリシーページは GitHub Pages を使用してホスティングされます。本ウェブページにアクセスすると、GitHub は GitHub 一般プライバシーステートメントに従って技術情報を処理する場合があります。本ページ自体には、AusRights が運用する分析、広告、またはトラッキングスクリプトは含まれていません。

子どものプライバシー

AusRights は、子どもを含め、いかなる人からも個人情報を意図的に収集しません。

情報の開示および販売

AusRights は個人情報を収集しないため、本アプリを通じて個人情報を販売したり、第三者に開示したりすることはありません。

本ポリシーの変更

AusRights の機能またはプライバシー上の取扱いが変更された場合、本プライバシーポリシーを更新することがあります。改訂版を公開する際は、上記の施行日を更新します。
